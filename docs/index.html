What is a Blockchain?

A blockchain is a distributed, immutable ledger that records transactions across many computers. This ensures that the record cannot be altered retroactively without the alteration of all subsequent blocks and the consensus of the network.

This demo will walk you through the fundamental concepts of blockchain technology:

  • Hash: Learn how cryptographic hash functions work
  • Block: Understand how data is stored in blocks
  • Blockchain: See how blocks are linked together
  • Distributed: Experience how a blockchain works across a network
  • Tokens: Learn about digital assets on a blockchain
  • Coinbase: Understand how new tokens are created
