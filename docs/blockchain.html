What is a Blockchain?

A blockchain is a chain of blocks linked together by including the hash of the previous block in the current block. This creates a chain of blocks that cannot be tampered with without invalidating all subsequent blocks.

Key features of a blockchain:

  • Previous Hash: Each block contains the hash of the previous block
  • Immutability: Changing data in one block invalidates all subsequent blocks
  • Chain Integrity: The entire chain can be verified by checking each block's hash
