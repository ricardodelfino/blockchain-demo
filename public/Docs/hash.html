What is a Hash?

A hash function is any function that can be used to map data of arbitrary size to fixed-size values. The values returned by a hash function are called hash values, hash codes, digests, or simply hashes.

In blockchain, we use cryptographic hash functions like SHA-256 which have these properties:

  • Deterministic: The same input always produces the same output
  • Fast to compute: It's quick to calculate the hash of any input
  • Pre-image resistant: Given a hash, it's computationally infeasible to find the original input
  • Small changes cause big differences: A tiny change in input produces a completely different hash
  • Collision resistant: It's extremely unlikely to find two different inputs that produce the same hash
