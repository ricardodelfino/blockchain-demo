What is a Block?

A block in a blockchain contains:

  • Block Number: The position of the block in the chain
  • Nonce: A number used once in the mining process
  • Data: The information stored in the block (transactions, etc.)
  • Hash: A unique identifier created from the block's contents

Mining a block means finding a hash that meets certain criteria (like starting with a specific number of zeros). This is done by changing the nonce until a valid hash is found.
